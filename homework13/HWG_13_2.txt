Требование: Завершение Pull Request
Модуль:  Git_Repo_PR
Имя: Pull Request
Предусловие/Setup: 
1.	В браузере перейти на страницу https://github.com/
2.	В верхнем правом углу нажать на кнопку «Sign in»
3.	Авторизоваться - ввести Login/Password , нажать «Sign in»
4.	Перейти во вкладку Repositories по центру в верхней части страницы
5.	Выбрать Репозитоий Main
6.	Перейти во вкладку branches 
7.	Создать новую ветку, нажатием на кнопку в правой верхней части страницы New Branch
8.	Ввести имя новой ветви и подтвердить его нажатием на кнопку Create Branch
9.	Создание отдельной папки и добавление файла для Pull Request, для этого нажимаем кнопку Add file на выпадающем списке нажимаем Create new file.
10.	В поле имени файла в верхней левой части экрана 
пишем название папки «homework13» ставим «/» и создаем файл test.txt

11.	Нажимаем Commit new file в нижней части страницы 
12.	Переходим на вкладку Pull Requests 
Шаги:
1.	Создать новый Pull Request нажатием на кнопку New pull request 
2.	Выбрать из какой ветки и в какую мы будем проводить объединение веток
 

3.	Создать Pull Request нажатием на кнопку Create Pull Request  
4.	Подтвердить создание Pull Request нажатием на кнопку Create Pull Request  

5.	Завершить объединение веток Git нажать кнопку Rebase and Merge 

Ожидаемый результат:
1.	После проведения Pull Request автоматический переход в основную ветвь main в которой должны появиться папки и файы из объединяемой ветви Mogilev_Nikita_HW13
